## Features - 產品功能

1. 使用者可以瀏覽全部的餐廳
2. 使用者可以點擊某餐廳瀏覽該餐廳的細節資訊
3. 使用者可以依照店名進行搜尋
4. 使用者可以依照餐廳類別進行搜尋
